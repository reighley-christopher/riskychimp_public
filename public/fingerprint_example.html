To use the flash fingerprinter you will need:

  1. A Form Field to transmit the data to the Merchant server. In this example fingerprint_field
  2. A fragment of javascript to copy the data to the form field when it is ready. In this example it is called check
  3. An embed tag with a url to the fingerprinter on the riskybiz server. In this example fingerprinter

Pay attention to the src attribute in the embed tag. It should be the url to the riskybiz server. The code should be downloaded from our server. Making a copy will not work.

The flashvars variable contains an "onload" key which will fire a javascript function when the data is ready.
